# CourseKata Teaching Resources Website Specification (2026)

*As a \< type of user \>, I want \< some goal \> so that \< some reason \>.*

# **Logged-In Site**

## **All Logged-In Users**

As any logged-in user, I want…

* to get to the place I’m interested in as quickly as possible so that I don’t waste time or get confused.  
* to search, sort, and filter resources so that I can efficiently find what I need.  
* resources to be clearly described and previewable so that I can quickly determine whether they meet my needs.  
* access to discussion and support so that I can ask questions and learn from others.  
* [Sample Class Log](https://docs.google.com/spreadsheets/d/1iZxrvHGbkonIDRgcQCw8iAQFx6kAreu_wmT00HB4CGs/edit?gid=1445774738#gid=1445774738)

## **Instructors**

As an instructor, I want….

* to ask AI to help me find what I need  
* to sort and filter resources by:  
  * **Type** (including JNB subtypes and other categories)  
  * **Function** (teaching, assessment, etc.)  
  * **Modality** (paper-based, online)  
  * **Coverage** (page, chapter, sections, book, concept)  
  * **Dataset**  
  * **Textbook version**  
    * **Identify version differences**  
  * Last updated  
  * Any additional tags added over time

so that I can quickly locate materials appropriate for my course.

* each resource to clearly display:  
  * Short description (e.g., “Students fit a model to characters data…”)  
  * What students actually do  
  * The learning goal(s) supported  
  * Length of time required  
  * Extra materials needed (e.g., scissors, glue) (Yes/No)  
  * Dataset(s) used  
    * The number of variables and the type of variables  
    * Topic tags to easily search different types of data  
  * Whether student data collection is required

so that I can confidently plan lessons.

* to preview a resource before using it so that I understand what will happen in my classroom.  
* to be able to download multiple resources at the same time based on some selection criteria so that I can quickly get all of the resources I want to use or explore further.  
* to access or learn about the source of the dataset (or how fake data was generated) so that I can answer student questions and ensure transparency.  
* to see evidence that students learn from and like the materials so that I feel confident investing class time.  
* to see indicators of adoption (e.g., number of downloads, number of teachers using it, reviews by other teachers) so that I can judge quality and trustworthiness.  
* to ask questions of CourseKata staff and other teachers so that I can get help when needed.  
* access to broader research and pedagogy (e.g., blog posts, summaries, academic papers) so that I understand the theory behind the design of the materials.

## **Site Administrator**

As a site administrator, I want the ability to:

* Add or remove resources  
* Add, edit, or rename tags  
* Create new sortable/filterable metadata categories  
* Edit resource descriptions and metadata

so that the system remains flexible, accurate, and responsive to user needs.

# **Not Logged-In Site**

## **All Logged-Out Users**

As a logged-out user, I want…

* to quickly get to the page I need so that I can save time.  
* to preview and understand what CourseKata resources are and who they are for so that I can decide whether to explore further.  
* to see trust signals (evidence of learning, testimonials, adoption indicators) so that I can evaluate credibility before creating an account.

# **Support (Logged In & Logged Out)**

As any user, I want an easy way to:

* Ask questions of CourseKata staff  
* Ask questions of other teachers  
* Participate in discussion spaces (e.g., Discourse)  
* Access help documentation

so that I can get timely support and feel part of a community.
